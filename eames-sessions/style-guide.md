# ListyGifty Design System Style Guide

## Color Palette

### Primary Colors
- **Primary**: `bg-violet-600` (#7C3AED), `bg-fuchsia-600` (#C026D3)
- **Primary Light**: `bg-violet-500` (#8B5CF6), `bg-fuchsia-500` (#D946EF) 
- **Primary Gradient**: `bg-gradient-to-r from-violet-600 to-fuchsia-600`

### Background Colors
- **Main Background**: `bg-slate-950` (#020617)
- **Card Background**: `bg-slate-900/50` (#0F172A with 50% opacity)
- **Section Background**: `bg-slate-900` (#0F172A)

### Text Colors
- **Primary Text**: `text-white` (#FFFFFF)
- **Secondary Text**: `text-slate-400` (#94A3B8)
- **Muted Text**: `text-slate-300` (#CBD5E1)
- **Accent Text**: `text-violet-400` (#A78BFA), `text-fuchsia-400` (#E879F9)

### Border Colors
- **Primary Border**: `border-slate-800` (#1E293B)
- **Hover Border**: `border-slate-700` (#334155)
- **Accent Border**: `border-violet-500/20` (#8B5CF6 with 20% opacity)

## Components

### Buttons
Primary CTA: `bg-gradient-to-r from-violet-600 to-fuchsia-600 hover:from-violet-500 hover:to-fuchsia-500 text-white shadow-xl shadow-violet-500/25`
Secondary: `border-slate-700 text-slate-300 hover:bg-slate-800 hover:text-white hover:border-slate-600`

### Cards
Feature Card: `bg-slate-900/50 border border-slate-800 hover:border-slate-700 transition-all duration-300`
Large Card: `bg-gradient-to-br from-slate-900 to-slate-800 border border-slate-700`

### Effects
Gradients: `bg-gradient-to-r from-violet-400 via-fuchsia-400 to-pink-400`
Shadows: `shadow-xl shadow-violet-500/25`
Transitions: `transition-all duration-300`
